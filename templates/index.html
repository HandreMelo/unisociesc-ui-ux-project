Como acessar os dados

Faça uma requisição GET para este endereço:

https://firebase-middleware-2.herokuapp.com/GAME

onde GAME é o código do jogo que deseja os dados dos jogadores (em Json).

Games cadastrados: Game1, Game2, Game3

Exemplo:

https://firebase-middleware-2.herokuapp.com/Game1

Retorno:

{
 "dHrXcloremoRoPZipsumMKPG3": //ID
 {
  "userName":"TheBest", //Nick do jogador
  "userScore":-1 //Score do jogador
 }
}
